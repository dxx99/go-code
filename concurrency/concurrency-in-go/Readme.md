## concurrency in go

### 介绍
- 概念介绍
    - 并发
    - 并行
    - 异步
    - 线程化
- 并发困难的原因
    - 数据竞争
        - 内存访问同步(访问同一内存区域)
    - 原子性
    - 锁的问题
        - 死锁：所有的并发进程都彼此等待
        - 活锁：正在主动执行并发操作，但这些操作无法向前移动
        - 饥饿：并发进程无法获得执行工作所需要的任何资源
