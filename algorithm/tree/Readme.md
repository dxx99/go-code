## 二叉树

### 递归
**三要数**
- 参数、返回值
- 终止条件
- 单层逻辑

**递归的本质**
- 每一次递归调用都会把函数的局部变量、参数值和返回值地址等压入调用栈中
- 返回时，从栈顶弹出上一次递归的各项参数，所以这就是递归可以返回上一层位置的原因
